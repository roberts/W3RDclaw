# W3RDclaw 🦀

open source secure local ai 4 [homelab](https://github.com/drewroberts/homelab) ... linux aur 🦀

---

### 🦀 overview

welcome to [W3RDclaw](https://github.com/roberts/W3RDclaw). this project is designed to provide a secure, local-first ai solution for your personal infrastructure. focused on long term asynchronous tasks from remote connections via ssh or thru blockchain call data.

[andyclaw](https://github.com/drewroberts/andyclaw) will handle remote handlheld devices.

### 🦀 features

* **privacy first**: all data stays within your local network
* **decentralized storage**: powered by the [W3RD](https://github.com/roberts/W3RD) database
* **homelab ready**: optimized for self-hosting enthusiasts

### 🦀 getting started

1. clone the repository:
`git clone https://github.com/roberts/W3RDclaw`
2. configure your [homelab](https://github.com/drewroberts/homelab) environment variables
3. initialize the [W3RD](https://github.com/roberts/W3RD) decentralized database
4. run the setup script to deploy your local ai
